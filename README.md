## Emotion
WebAPIを使って画像の表情認識を行います．

①　使用する画像ファイルを`ファイルを選択`のボタンから読み込みます（png,jpg）  
②　読み込まれた画像ファイルが表示されます．  
③　レーダーチャートに`Cloud Vision API`の結果が表示されます．  
④　レーダーチャートに`FACE API`の結果が表示されます．  
⑤　chromeの開発ツールを用いて，APIから受け取った値を見ることができます．
